Логотип
Логотип сервиса можно использовать только белым на акцентном фоне и ацентным на белом.
Логотипом считается знак + надпись My Wish, по вопросам пишите в отдел дизайна коммуникаций
Кнопка
mw-button
Кнопка. Используется, когда человеку нужно сделать выбор, подтвердить или совершить какое-то действие. Например, отправить сообщение, оплатить заказ, сохранить изменения, войти в акканут. 
<button class="mw-button mw-button_size_m mw-button_view_primary mw-button_has-icon_no-icon">...</button>
Чипс
mw-chips
Выполняет фильтрацию контента, работает по принципу радиокнопки. В интерфейсе обычно используется в количестве 2+ 
<div class="mw-chips mw-chips_size_m">...</div>
Чекбокс
mw-checkbox
Чекбокс — элемент с двумя состояниями: выбран и не выбран. Используйте чекбокс, если хотите, чтобы пользователь сделал выбор.
